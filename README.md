# Blogify – Blogging Web Application (CRUD)

 Project Overview
Blogify is a full-stack blogging web application that allows users to create, read, update, and delete blog posts.  
The application is built using **React (Frontend)**, **Node.js + Express (Backend)**, and **MongoDB (Database)**.

The project demonstrates full CRUD functionality using REST APIs with proper frontend–backend integration.

---

 Features
- Create new blog posts
- View all blog posts
- Edit existing blog posts
- Delete blog posts
- User-friendly and responsive UI
- RESTful API integration
- MongoDB database storage
- Login & Register (frontend-based)
